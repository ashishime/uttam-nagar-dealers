## ✨ Features

- 🌑 Premium dark-mode UI with glassmorphism design
- 📱 Fully responsive (mobile, tablet, desktop)
- 🔍 Filter by top-rated or most reviewed dealers
- 📞 One-click call buttons
- 🗺️ Google Maps integration for directions
- 📬 Contact enquiry form
- ⚡ Animated counters, particles and card reveals
- 🔢 SEO optimized (meta tags, semantic HTML, structured data)

---

## 🛠️ Tech Stack

- **HTML5** – Semantic structure
- **Vanilla CSS** – Custom properties, glassmorphism, animations
- **Vanilla JS** – IntersectionObserver, particle system, filters
- **Data** – Google Maps scraped via [Apify crawler-google-places](https://apify.com/compass/crawler-google-places)
- **Hosting** – GitHub Pages
